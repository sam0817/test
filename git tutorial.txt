
1. Git 提交
  a. Stage 暫存
  b. Commit  提交
    * 訊息明確清楚
	* 小範圍多次提交
  c. Diff 
    * Working - Stage
	* Stage - Commit
	* Commit - Commit
  d. HEAD 
  e. SHA-1 node
  
2.  Git Branch
  a. Checkout 提取 (node, file)
  c. Conflict 衝突
  b. Merge 合併
  c. Rebase 合併為同一條線
  d. Reset 重設 (hard rollback)
  e. Revert 還原 (soft rollback)
  f. Cherry Pick 挑選
  

3. Remote
  a. Fetch 取得遠端儲存庫
  b. Pull (Fetch + Merge)
  c. Push (遠端提交不可修改，需先合併才能上傳)
