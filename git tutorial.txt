
1. Git 提交
  a. Stage 暫存
  b. Commit  提交
    * 訊息明確清楚
	* 小範圍多次提交
  c. Diff 
    * Working - Stage
	* Stage - Commit
	* Commit - Commit
  d. HEAD 
  e. SHA-1 node
  
2.  Git Branch
  a. Checkout 提取 (node, file)
  c. Conflict 衝突
  b. Merge 合併
  c. Rebase 調整提交順序
  d. Reset 重設 (hard rollback)
  e. Revert 還原 (soft rollback)
  f. Cherry-Pick 挑選特定點
  

3. Remote
  a. Fetch 取得遠端儲存庫
    * 遠端為獨立倉儲，需人工合併本地端
  b. Pull (Fetch + Merge)
    * --rebase
  c. Push 
    * 遠端被更新
  d. rebase vs merge
    * clean vs ugly
	* origin vs modified
  
4. Other Features
 a. Stash (打包)
 b. Tag (commit alias only)
 c. Blame (記錄)
 d. 相對指標 ~ ^
